# React 仿写简书网站

> 这是一个学习react的项目  

项目借实现简书网站ui，学习react开发及周边主流工具使用，主要实现了首页ui，简略实现了文章详情及login的ui开发。


| 项目涉及的主要工具库 | 作用 | 
| ------ | ------ | 
| react | 基础ui库 | 
| react-redux，redux | 用于数据状态存储 | 
| react-router-dom | 用于路由 | 
| react-transition-group | 用于动画过度 | 
| immutable | 用于封装js数据 | 
| styled-components | react下CSS解决方案 | 
| redux-thunk | 使action支持函数 |
